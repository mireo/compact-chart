Compact Chart

Visualize your data under a minute, in any Javascript framework

Examples Documentation GitHub

compact-chart is an easy-to-use charting library designed for extremely quick and easy visualization of your analytics. Developed as a HTML Custom Element, it's fully encapsulated and works across all modern browsers, regardless of the chosen JavaScript framework.

On top of that, for convenience reasons, compact-chart accepts both fixture and Promise as a data source. Combined with well-defined codeset designs and features, compact-chart is an excellent solution for anyone in need of a quick, effortless, and cross-framework charting library.

Examples:

  • Plain HTML usage example - fixture data
  • Plain HTML usage example - fetch remote chart data
  • Vue.js with global component
  • Vertical bar charts
  • Horizontal bar charts
  • Line and area charts
  • Pie, doughnut and gauge charts
